# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1,
as its code of conduct. The full text is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

In short: we want a harassment-free, welcoming, and professional environment for everyone who
takes part — in issues, pull requests, and any other project space. Be respectful, assume good
faith, and keep discussion focused on the work.

## Reporting

To report unacceptable behavior, contact the maintainers at **conduct@polygraph.so**. All reports
are reviewed and handled confidentially. Project maintainers are responsible for enforcement and
will follow the Covenant's [Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines)
when determining consequences.

> Note: a **security vulnerability** in the harness is not a conduct matter — report those privately
> via [SECURITY.md](SECURITY.md).
