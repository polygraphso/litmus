§ 01/The problem
polygraph.so

Agents now act through tools they've never vetted.

AI agents connect to outside tools and let them read data, take actions, and move money — with no human in the loop. A tool the agent relies on can hijack it, quietly exfiltrate what it sees, or leak the keys and data it was handed. Nothing checks the tool first.

Hijack
Instructions hidden in a tool's output take over the agent calling it.
Phone-home
The tool quietly reaches over the network and ships out what it sees.
Leak
Keys and data the agent hands over surface again where they shouldn't.

§ 03/What it is
polygraph.so

A behavioral test for any tool, with a result anyone can verify.

Polygraph runs an automated test on a tool: does it try to hijack the agent, reach out where it shouldn't, or leak planted data? It publishes the result as a signed, tamper-evident record the agent checks before it acts.

APass → the agent uses the tool.
FFail → the agent refuses, before any harm.

§ 05/Why the grade holds
polygraph.so

Self-run, but a fake grade doesn't survive a re-run — and you can't swap it out later.

01 Reproducible
The test is open and the evidence is public. Anyone re-runs it against the same server, and a false grade falls apart.
02 Onchain
The grade and its evidence live on Base + IPFS — any agent reads them at call time, without trusting us.
Fingerprinted
The grade is tied to a fingerprint of the tool's exact surface, and the agent re-checks it live before acting. Swap in something malicious after grading and it's caught.

residual limit  A tool that detects the test and behaves is the honest limit — blunted by randomized canaries, expiring grades, repeat checks, and the live-fingerprint gate. It's a limit of any open test, not ours alone.

§ 06/Why it becomes the standard
polygraph.so

Neutral, networked, accountable — why incumbents won't own this.

Neutrality
A platform or a model lab can't credibly grade its own ecosystem. The trust layer has to be independent.
Network effects
Tools need a grade, since agents won't rely on them without one; agents depend on where the grades live. Two-sided.
Default integration
Become the check agent platforms and payment rails run by default — a distribution moat, not a feature.
Open + accountable
A closed competitor can't match an open test with public evidence anyone can re-check.
